Projects

Typopo
Good luck getting on the leaderboard.
Typopo is an interactive words-per-minute typing game with leaderboard updated in real time and a chat room feature. Inspired intellectually by Mario Teaches Typing and visually by the net.art movement, Typopo was built using React components to continuously re-render text based on correct input and to calculate scores. Typopo uses a Postgres database to store and update new high scores, and utilizes web sockets for chat functionality, with help from Socket.iO.

Play Typopo here or visit the code on GitHub here.
Rate My Landlord
Rate My Landlord is a Progressive Web App that allows users to search NYC buildings and landlords, leave reviews, expore and visualize housing-related data, contact other users, and view curated rental listings.
RateMyLandlord homepage.
A page containing all the data and reviews for a single landlord.
Inspired by the popular website Rate My Professor, Nuala and her co-developers hoped to create a useful resource that included relevant features for those moving within or to New York, while also using the project as an opporunity to explore new technologies. For the "explore" page, Nuala and her teammates played around with data visualization libraries like Chart.js On the account home-page view, for a logged-in user, the team built in a curated rental listing feature based on user preferences using web-scraping with Puppeteer.js.
Completed by a team of four over a 12-day sprint, Rate My Landlord is built with Node.js, React, Redux, Express, Postgres, Sequelize, Chart.js, Materialize UI, and Puppeteer.

Go ahead and rate your own landlord here, check out out the code on GitHub here, and watch the group's capstone presentation on YouTube here.
A logged-in user's account home-page view.
Bubblysort
Bubblysort homepage
Bubblysort is a mock e-commerce site dedicated to selling fake seltzer. Nuala and her team of 4 developed Bubblysort over a 5-day sprint.
Bubblysort provides a seamless shopping experience for new and returning users, with persistent cart functionality, checkout, and Google oAuth options.
